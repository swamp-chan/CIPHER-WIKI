ABOUT

The purpose of the creation of this website is primarily for the showcasing of practical application of responsive design by using HTML and CSS. By utilizing the combination of HTML5 and modern CSS, like flexbox and grids, information can be given to end users in a appealing manner. Ulimately this project serves to fullfill the grade 11 requirement for a CS project work.

Beyond the school driven HTML-CSS showcase, this project explores some preliminary concepts present in the field of cybersecurity, mainly cryptology. The very basics of cryptology's information is explained in the INTRODUCTION tab. If you wish to learn about basic cryptography, do check it out.

Each and every tool or "inspiration" which has aided me in this making of this project has been appropriately referenced in the REFERENCE tab.
